Tentang Saya

Perkenalkan nama saya Armasnyah Aris Nugraha saya memulai jalur karir dari latar belakang SMK Manajemen Perkantoran (MPLB), lalu secara mandiri mendalami pengembangan web selama dua tahun terakhir secara otodidak. Ketelitian administratif yang saya bangun bertahun-tahun kini tercermin dalam cara saya menyusun kode: terstruktur, terdokumentasi, dan mudah dipelihara.

Perjalanan dari spreadsheet ke source code bukan sekadar perubahan alat — ini adalah transformasi cara berpikir. Saya belajar bahwa logika di balik formula Excel dan logika di balik fungsi JavaScript ternyata tidak jauh berbeda: keduanya tentang mengolah data menjadi hasil yang bermakna.

Pendidikan

SMK Negeri — Manajemen Perkantoran dan Layanan Bisnis (MPLB), lulus 2023.

Tujuan Karir

Saya ingin menjadi seorang junior frontend developer yang handal dan dapat dipercaya oleh perusahaan, khususnya di bidang pengembangan web dan kecerdasan buatan (AI). Serta tertarik membangun produk yang memecahkan masalah nyata.
